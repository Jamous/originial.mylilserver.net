WVRU Catalog server

WVRU, Radford University's public radio station, has a jazz cd colection of roughly 3,000 CDs. At this time the majority of the CDs are cataloged, but that catalog is spread across several excel documents, making the catalog virtualy imposible to search. The prupose of this project are as follows.
  • Hold the Artist name, Album Name, Record Lable, Year released, genera, and a short discreption of the song in an SQL database.
  • Write a program to take the existing records, convert them into a relational database friendly format, and upload them to the database.
  • Be easily able to querry the database.
  • Build a Web interface that can query the database. The end user should never have to use any SQL comands.
  • Build a managment front end that allows an end user to add and edit entries without having to use SQL comands.
  • Privately host the system on a Ubuntu 18.04 LTS platform, allowing me to pass it on to Radford Universitys IT department, once i am done building it, to permantly deploy it.
While there is currently nothing on the website, you can access that machine at wvru.mylilserver.net.
I will update this page with my progress on that project.
